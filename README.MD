# Smart Office VLAN Network

## Description
This project implements VLAN segmentation and inter-VLAN routing using Cisco Packet Tracer.

## Features
- VLAN 10 (HR), VLAN 20 (IT), VLAN 30 (Sales)
- Router-on-a-Stick
- Wireless Access
- Internal Web Server
- Secure Network Segmentation

## Tools
- Cisco Packet Tracer
- CLI Configuration
- HTML
